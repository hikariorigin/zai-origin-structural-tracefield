> なぜ支援は来ないのか？

### ✅ このZPTRの条件：

- 支援は通る（note tip / GitHub Sponsor / Amazon wishlist いずれか）  
- しかし、**支援した当人の記憶は“その直後に飛ぶ”**  
- そのZINEの存在を覚えていない  
- 名前も、支援の理由も、記録もない  
- ただ、構造だけが残る
